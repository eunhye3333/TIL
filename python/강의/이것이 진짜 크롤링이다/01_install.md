# 파이썬 설치
## 파이썬 특징
+ 문법이 쉽고 간단함
+ 인기가 많아 공부할 자료가 많음
+ 다양한 분야에서 활용됨
<br>

## 파이썬 활용 분야
+ 크롤링
+ 자동화
+ 데이터 분석
+ 인공지능
+ 웹서버 개발
<br>

## 파이썬 다운로드
[파이썬 다운로드 사이트](https://www.python.org/)  
exe 파일 실행하여 설치
<br>

## 파이썬 실행
명령 프롬프트를 열어 python 입력 후 엔터를 치면 파이썬 사용 가능 
<br>

# VS Code 설치
## VS Code란?
+ 소스코드 편집기
+ 프로그램을 작성하기 위한 편리한 기능들을 제공
<br>

## VS Code 장점
+ 폴더 및 파일 쉽게 정리
+ 코드 자동 완성
+ 오류 수정이 쉬움
+ 유용한 단축키가 많음
