# 크롤링을 위한 정말 쉬운 HTML
## HTML
+ 웹 페이지의 **구조**를 나타내기 위한 언어
+ 태그로 구성

## 태그
+ 웹페이지의 구성 요소로 <>로 표시
+ 태그는 시작 태그와 종료 태그로 만들어짐  
  ex. <태그명> 내용 </태그명>
+ 태그에는 속성명과 속성값이 있을 수 있음  
  ex. <태그명 속성명="속성값"> 내용 </태그명>
+ 크롤링에 자주 사용되는 태그 종류  
  + div : 구역 나누기 (제목, 본문 등의 구역)
  + a : 링크
  + h1 : 제목
  + p : 문단
  + ul, li : 목록
+ 태그는 부모 태그와 자식 태그가 있음
